About Mallochite

Mallochite is an open source peer-to-peer chat application. With Mallochite you can exchange messages with other users of Mallochite! You also are able to send and recieve the photos, philes, packages via our data exchange module, implemented in the application! You can make voice calls, video calls, send emojis and many many other things! Now, you will say there are a lot of applications having same features. How is this different? And the Answer for that could be as simple as "We are just more awesome than all of those suckas" But in fact Mallochite is completely different from other applications providing same array of features because it is safer. You can't see it but it is there. Mallochite is using completely different networking paradigm which does not require servers which usually collect data from users. No such things happen in Mallochite's back-end. We do not collect any forms of data from users. We do not store the messages you send on remote server databases. All data traveling through the network is encrypted and signed. There are only two users who can see your messages: You and whoever you are chating with.
